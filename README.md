## 🏡 git-3D-garden

A 3D github's contribution activities visualized out as the 3d tile set

## 🚀 Getting Started

1. **Start the react-app.**

###### start the react app in the development mode with 

```sh
yarn start
```

2. **Build project.**

###### build the react app

```sh
yarn build
```
